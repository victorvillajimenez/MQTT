# MQTT

## About

The purpose of this repo is to understand MQTT (Message Queuing Telemetry Transport), and how we can create applications that can communicate each other using an MQTT Broker, in this case HiveMQ Cloud.

Projects:

- iot-mqtt-poc (IoTMqttPoC)
- iot-mqtt-pubsub-server (IoTMqttPubSubServer)
- iot-mqtt-pubsub-client (IoTMqttPubSubClient)
- IoTMqttPLC (IoTMqttPLC)
